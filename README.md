# 🏥 AI Medical Assistant RAG

A Retrieval-Augmented Generation (RAG) system that allows users to upload medical documents and receive context-aware AI-generated answers.

## 📌 Overview

This project combines document retrieval and Large Language Models to create a medical knowledge assistant capable of answering healthcare-related questions from uploaded documents.

---

## ✨ Features

- PDF Upload
- Document Parsing
- Embedding Generation
- Vector Search
- Retrieval-Augmented Generation
- Context-Aware Responses

---

## 🛠️ Tech Stack

### Backend
- Python
- FastAPI

### AI
- LangChain
- Sentence Transformers
- ChromaDB

### LLM
- Groq API
- Llama Models

### Database
- Chroma Vector Database

---

## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/AishwariyaAI/AI-Medical-Assistant-RAG.git
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
uvicorn app.main:app --reload
```
